# TSB-DebugArtifact
TSBにデバッグ用神器を追加する二次創作。  
扱いを誤ると不具合に繋がる可能性があるため、自己責任で使用してください。


## 主な内容
以下の4つのデバッグ用神器を追加します。  

<details>
<summary>滅虫器 [α]</summary>

|プレイヤーの状態|効果|
|---|---|
|直立|MPを9999回復|
|直立+下向き|MPを9999消費|
|スニーク|HPを9999.9回復|
|スニーク+下向き|自身のHPを瀕死まで減らす|
|スニーク+上向き|開発特権[^1]の取得/剝奪|

[^1]: 信仰と残りMPを無視して神器を使えるようになる。
</details></br>

<details>
<summary>滅虫器 [β]</summary>

|プレイヤーの状態|効果|
|---|---|
|直立|難易度を切り替える|
|直立+下向き|難易度を限界突破[^2]させる(連続使用可能)|
|直立+上向き|信仰を切り替える|
|スニーク|無敵化の切り替え|
|スニーク+下向き|半径50m内の全ての敵に9999.9ダメージを与える|

[^2]: TSBの難易度は1~3の数値で管理されており、この数値によって受けるダメージやデバフのスタック数などが制御されています。</br>つまり、限界突破させることでこの数値を4以上にすることが可能です。
</details></br>

<details>
<summary>滅虫器 [γ]</summary>

- スニークをし続けることで、近くの浄化済みの呪われた神器を再び呪う。  
この時かつて天使がいた場合は復活させ、再戦することができる。
</details></br>

<details>
<summary>デバッグ用コマンド説明書</summary>

- <span style="color: red; ">※この神器はコマンドを実行できる権限が無いと使用できません。</span>

- 説明書内に書かれた文字をクリックすると、  
指定した神器の取得やモブの召喚などが行えるデバッグ用のコマンドの使い方が表示され、  
<span style="color: orange; ">[例文をチャット欄に入力]</span>をクリックするとコマンドがチャット欄に入力された状態になる。  
</details></br>

## 各対応Verや導入条件
Minecraft: v1.20.4  
The Sky Blessing: v1.0.6  
途中導入: 可能

## 導入方法
1: Releasesからデータパックをダウンロードする  
2: 既に古いVerの物を導入している場合は削除しておく  
3: 導入したいワールドのdatapacksフォルダにダウンロードしたデータパックを入れる(解凍不要)
